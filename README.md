# JS Playground

This is a beginner-friendly JavaScript practice demonstration from [BroCode's 12-hour JavaScript tutorial](https://youtu.be/lfmg-EJ8gm4?si=cHtiZEi7pS3Kk4Ny) on YouTube. The practice includes tasks covering the fundamental concepts of JavaScript, such as variables, functions, DOM manipulation, and event handling.

This design was created specifically to originally demonstrate the practice tasks. The original Figma file can be viewed [here](https://www.figma.com/design/gMgjYdlvnTfxuQdSbGjo5G/JS-playground?node-id=0-1&t=YLe7LDGMty3TTB0D-1).
